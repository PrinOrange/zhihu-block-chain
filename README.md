# ZHIHU BLOCK CHAIN

这是一个用于知乎网站的一键拉黑工具。它支持以下三种工作方式：

1. 拉黑所有 **赞同指定文章、回答的用户**
2. 拉黑指定用户的 **所有关注者**
3. 拉黑指定用户所有 **正在关注的人**
